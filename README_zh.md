# 你好，我是 Vocsal

我目前主要专注于 **AI 应用全栈工程**，把 AI 能力做成真正可用、可落地的产品体验。

我的工作覆盖从产品原型、前端体验、后端服务，到模型/API 集成、自动化流程和工程交付的完整链路。相比单纯调用模型，我更关心 AI 如何进入真实业务流程，并稳定地为用户解决问题。

[English Version](./README.md)

## 我在做什么

- AI 原生 Web 应用和内部工具
- 从想法到可用 Demo 的全栈产品原型
- 基于 LLM 的工作流、Agent 和自动化系统
- 使用 React、TypeScript 和现代前端工具构建产品界面
- Node.js 服务、脚本、爬虫和数据处理工具
- Three.js / WebGL 方向的创意 Web 3D 实验

## 当前关注方向

- 构建体验扎实、真正有用的 AI 应用
- 将 LLM 接入真实产品流程和业务逻辑
- 设计可靠的工具调用、检索增强和自动化模式
- 探索 AI 辅助内容生产和短视频工作流
- 让界面保持简单、快速、可理解
